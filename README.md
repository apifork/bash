# [bash.apifork.com](https://bash.apifork.com/)

## Start

apifork implementation in project

```bash
curl https://github.com/apifork/bash/apifork.sh
echo "apifork.txt" > ".apifork"
echo "" > "apifork.txt"
```

### install

```bash
./apifork.sh install
```
OR

```bash
./apifork.sh
```

### update

```bash
./apifork.sh update
```


### remove

```bash
./apifork.sh remove
```

Old version was working in separated files:

#./apifork/${CMD}.sh
