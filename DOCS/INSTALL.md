
## Install apifork [<span style='font-size:20px;'>&#x270D;</span>](https://github.com/apifork/examples/edit/main/DOCS/INSTALL.md)


### install

```bash
./apifork.sh install
```
OR

```bash
./apifork.sh
```

### update

```bash
./apifork.sh update
```


### remove

```bash
./apifork.sh remove
```
